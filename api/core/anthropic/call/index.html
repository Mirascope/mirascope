    mirascope.core.anthropic.call¶

    A decorator for calling the Anthropic API with a typed function.

    Usage Documentation

    Calls

    This decorator is used to wrap a typed function that calls the Anthropic API. It parses the prompt template of the wrapped function as the messages array and templates the input arguments for the function into each message's template.

    Example:

    from mirascope.core import prompt_template
from mirascope.core.anthropic import anthropic_call


@anthropic_call("claude-3-5-sonnet-20240620")
def recommend_book(genre: str) -> str:
    return f"Recommend a {genre} book"

response = recommend_book("fantasy")
print(response.content)
