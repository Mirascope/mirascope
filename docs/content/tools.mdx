# Tools

This document covers tool calling - giving LLMs the ability to invoke functions.

## Topics Covered

- Introduction to the concept of tool calling
- The `@llm.tool` decorator for defining tools
- Tool schemas: how function signatures and docstrings become tool definitions
- The `llm.ToolCall` content type in responses
- Checking for tool calls via `response.tool_calls`
- Executing tools with `response.execute_tools()`
- Providing tool outputs back to the LLM with `response.resume()`
- The `llm.Toolkit` concept for grouping related tools
- Async tools and `llm.AsyncTool`
