# Structured Output

This document covers generating structured output from LLMs using format classes.

## Topics Covered

- Introduction to structured output and why it's useful
- Format classes: Pydantic `BaseModel` subclasses that define response structure
- Passing a format via the `format=` parameter
- Using `llm.format()` to specify formatting modes (`strict`, `json`, `tool`, `parser`)
- How different formatting modes work under the hood
- Parsing responses with `response.parse()`
- Partial parsing during streaming with `response.parse(partial=True)`
- The `llm.Partial` type for streaming structured output
- Validation and error handling with Pydantic
- Custom output parsers for non-standard formats (XML, regex, etc.)
