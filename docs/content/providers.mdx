# Providers & Supported Features

This document covers provider configuration and feature support in Mirascope.

## Topics Covered

- Built-in providers: OpenAI, Anthropic, Google, and others
- Provider registration with `llm.register_provider()`
- Customizing provider settings: base URL, API keys, clients
- Routing models to providers (e.g., OpenAI-compatible APIs)
- Supported features matrix across providers
- Resetting provider configuration with `llm.reset_provider_registry()`
